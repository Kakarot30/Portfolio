## 📖 Overview

This project is a feature-rich, visually stunning portfolio template built with the latest web technologies. It is designed for developers, designers, and creatives who want to stand out with a modern, animated, and highly customizable portfolio.

---

## ✨ Features

- **Modern UI/UX**: Clean, responsive, and accessible design
- **Animated Interactions**: Framer Motion & Three.js for smooth, engaging effects
- **Bento Grid & Spotlight Hero**: Unique layouts and dynamic backgrounds
- **Project & Experience Sections**: Highlight your best work and career
- **Testimonials & Social Proof**: Build trust with dynamic testimonials
- **Dark Mode**: Seamless light/dark theme support
- **Easy Customization**: Modular components and data-driven content
- **Production Ready**: SEO, performance, and accessibility best practices

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
git clone https://github.com/adrianhajdin/portfolio.git
cd portfolio
npm install # or yarn install
```

### Development

```bash
npm run dev # or yarn dev
```

Visit [http://localhost:3000](http://localhost:3000) to view your portfolio.

---

## �️ Customization

- **Edit content**: Update `data/` and `components/` to personalize your projects, experience, and skills.
- **Change images/icons**: Replace assets in the `public/` folder.
- **Adjust styles**: Modify `globals.css` or Tailwind config for branding.
- **Add new sections**: Use the modular component structure to expand easily.

---

## 🌐 Deployment

Deploy to Vercel, Netlify, or any platform supporting Next.js. For Vercel:

```bash
npm run build
npm start
```

## 🎨 Theming & Gradients

Use Tailwind's utility classes or custom gradients for backgrounds:

```css
.black-gradient {
  background: linear-gradient(90deg, #161a31 0%, #06091f 100%);
}
```

Or in JSX:

```js
style={{
  background: 'rgb(4,7,29)',
  backgroundColor: 'linear-gradient(90deg, rgba(4,7,29,1) 0%, rgba(12,14,35,1) 100%)',
}}
```
