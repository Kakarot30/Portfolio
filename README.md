## üì¶ How to Use

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Kakarot30/Portfolio.git
   cd Portfolio
   ```
2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   # or
   yarn dev
   ```
4. **Open your browser:**
   Visit [http://localhost:3000](http://localhost:3000)
